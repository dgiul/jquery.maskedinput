# Download
Use `bower install jquery-number-mask` or [zip](https://github.com/Mavrin/maskInput/zipball/master)

# Build this project.
1. Install [node](http://nodejs.org/).
2. Install npm bower
    ```
          npm install -g bower
    ```
3. Get dependency
    ```
          bower install
    ```
4. Install gulp and gulp plugins
    ```
            npm install
    ```
5. Run gulp
    ```
            gulp
    ```
